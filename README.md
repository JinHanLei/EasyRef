# EasyRef——文献综述撰写工具

## 产品目标
自动爬取、总结、分类论文，生成bib引用，辅助撰写文献综述、文献阅读。

##  产品设计
输入关键词-输出爬虫获取的n篇论文的思维导图（mermaid）+总结+Reference。
